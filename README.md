## 🤝 介绍

使用A2S协议查询服务器信息，以图片或文字方式发送

## 📦 安装

- pip安装插件目录下的requirements.txt
- （如果只使用/ipt可以跳过此处）装完之后。进入到你`Astrbot`的环境下进行安装浏览器驱动，命令如下：
  ```shell
  playwright install chromium

## ⌨️ 使用说明

### 命令

```plaintext
/ip 地址:端口 （返回服务器信息图片）
/ipt 地址:端口 （返回服务器信息文字）
```
